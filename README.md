# SharingMyWishList

2022 밀리그램들의 프로젝트, **Sharing WishList** 입니다.<br/>
가입한 회원들에 한정하여 서로의 위시리스트를 작성, 공유, 달성 및 수정을 할 수 있습니다.

## Description

---

자신의 목표, 소원을 다른 사람들과 나누어 본 적 있나요? 사소한 것이라도 모두가 함께 나눈다면 행복은 배가 될 거에요.<br/>
평소 가지고 싶던 작은 것들, 이루고 싶던 작은 소원들을 적어보세요.
